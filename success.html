Frequently Asked Questions

Does this theme require any plugins?



No, there are no plugins required by this theme. Everything you need to design an amazing website is provided by the framework.

Does the Single site license allow development sites?



Yes, local and staging development sites are allowed to use the Single license. You will receive an extra development license free to use on your development sites.

Can I use my own child theme?



Yes, it is recommended to always use a child theme with Flexblox. A child theme is a safe way to customize your WordPress site without losing your changes due to theme updates. Flexblox provides multiple child themes to help you get started, or start from scratch using a free plugin like Create Block Theme.

How do I create a child theme?



We recommend using the create block theme plugin to create your own child themes. With a little know-how, they can be the perfect way to build your own website. They inherit all of the parent theme’s design which saves you a tonne of work and provides a head start for your next project.

Lightweight full site editing block theme framework. Flexblox FSE Theme provides useful features that help you build modern WordPress sites with blocks.
